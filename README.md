## 使用方法

1. 開始する`単語`を1つ入力する。

2. 入力した単語の末尾からAIが単語を1つ返す。

3. `1〜2`を勝敗が決まるまで繰り返す。


- 勝敗の判定方法

   -　単語の末尾が「ん」もしくは「ン」の場合。

   -　１度使用した単語を再度使用した場合。
